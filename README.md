# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data and ending with actionable business insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, interactive dashboard creation, report generation, and presentation preparation.

The objective of this project is to showcase practical data analytics skills using industry-standard tools and techniques.

---

## Dataset

The dataset contains business-related data used for analysis and visualization.

Key activities performed:

* Loaded the dataset using Python
* Explored data structure and quality
* Cleaned missing values and duplicates
* Prepared the data for SQL analysis and visualization

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and overall structure.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends.
* Identified missing values and outliers.
* Generated visualizations for better understanding.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into PostgreSQL, MySQL, and SQL Server.
* Wrote SQL queries to:

  * Filter records
  * Aggregate data
  * Perform joins
  * Calculate business metrics
  * Generate analytical insights

### 5. Power BI Dashboard

* Built an interactive dashboard.
* Created KPIs and charts.
* Added filters and slicers for dynamic analysis.
* Visualized key business insights.

### 6. Report & Presentation

* Documented findings in a project report.
* Created a presentation using Gamma.
* Finalized a PowerPoint summarizing objectives, analysis, insights, and recommendations.

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Load the dataset.
5. Run the notebooks in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
   * SQL Analysis
6. Open the Power BI dashboard (`.pbix` file).
7. Review the report and presentation files.
